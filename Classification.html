2. Logistic Concepts

Why do we call it Logistic Regression even though we are using the technique for classification?

Explain. Logistic regression is named that because it models the probability using the logistic function. Although it can be used for classification tasks, it is typically a regression model used to estimate the probabilities rather than predicting using discrete class labels directly.

3. Modeling

We train a logistic regression algorithm to classify a whether a wine comes from Marlborough using:

  1. An 80-20 train-test split.
  2. Three features engineered from the description
  3. 5-fold cross validation.

We report Kappa after using the model to predict provinces in the holdout sample.

4. Binary vs Other Classification

What is the difference between determining some form of classification through logistic regression versus methods like \(K\)-NN and Naive Bayes which performed classifications.

Explain.

The difference between determining some form of classification through logistic regression versus other methods is that logistic regression uses a linear function of the predictors to model the probability of a class and applies a threshold to classify instances. On the other hand, something like KNN is a non-parametric method that will assign a class based on the majority vote of its nearest neighbors. Logistic regression is also more interpretable for linearly separable data while the others are more flexible but may require careful tuning and assumptions.

Explain.

The ROC curve visualizes the trade-off between true positive rate/sensitivity and false positive rate at different classification thresholds. A model with a curve closer to the top-left corner and the area under the curve being close to 1 is considered a strong classifier.
