Feature Engineering

We begin by engineering an number of features.

  1. Create a total of 10 features (including points).
  2. Remove all rows with a missing value.
  3. Ensure only log(price) and engineering features are the only columns that remain in the wino dataframe.
wino <- wine %>%
  mutate(
    lprice = log(price),  #log transformation of price
    is_premium = ifelse(price > median(price, na.rm = TRUE), 1, 0),  # binary: premium wine
    age = 2025 - year,  #computes wine age
    high_quality = ifelse(points >= 90, 1, 0),  # binary: high quality
    province_encoded = as.numeric(as.factor(province)),  # encodes province as numeric factor
    region_encoded = as.numeric(as.factor(region_1)),  # encodes region_1 as numeric factor
    winery_encoded = as.numeric(as.factor(winery)),  # encodes winery as numeric factor
    variety_encoded = as.numeric(as.factor(variety)),  # encodes variety as numeric factor
    description_length = nchar(description),  # calculates the length of the wine description
    word_count = str_count(description, "\\s+") + 1  # word count in description
  ) %>%
  drop_na() %>%  #removes missing values
  select(lprice, points, is_premium, age, high_quality, province_encoded, region_encoded, 
         winery_encoded, variety_encoded, description_length, word_count)  #keeps only selected features

I have added in comments next to each line to describe what it is doing.

Caret

We now use a train/test split to evaluate the features.

  1. Use the Caret library to partition the wino dataframe into an 80/20 split.
  2. Run a linear regression with bootstrap resampling.
  3. Report RMSE on the test partition of the data.
# Train/Test Split
#set.seed(5)  
train_index <- createDataPartition(wino$lprice, p = 0.8, list = FALSE)  
train_data <- wino[train_index, ]
test_data <- wino[-train_index, ]

Splits the dataset into 80% training and 20% testing based on lprice.

# Train Model using Linear Regression with Bootstrap Resampling
model <- train(lprice ~ ., #lprice ~ points + winery_encoded,
      data=train_data,
      method="lm",
      trControl = trainControl(method = "boot", number = 100))

I am first making a split for my training and testing data from the new wino dataset. I then create a model using the bootstrap resampling and our train_data.
